Quick Start
1. Start service (default): http://127.0.0.1:5000
2. Configure model access: copy .env.example to .env, fill VOLCANO_API_KEY, then restart.
3. Run tasks in app; ARIA will match/learn methods and update long-term memory.

Why ARIA
ARIA binds task execution with reusable methods. Similar tasks reuse existing methods first, then new execution results are accumulated to improve later matching.
