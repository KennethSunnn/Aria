Computer control & automation

Without optional drivers, browser_click/browser_type and desktop_hotkey/desktop_type are simulated. Set ARIA_PLAYWRIGHT=1 and install Playwright + Chromium for real in-page actions; set ARIA_DESKTOP_UIA=1 on Windows with pywinauto for experimental hotkeys/typing. See README “Automation capability matrix” and .env.example.

You can also enable this with ARIA_ACTION_SCREENSHOT=1 in .env (either setting is enough).
